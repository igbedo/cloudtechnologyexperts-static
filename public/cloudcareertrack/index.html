Program Structure

Cloud Career Track Curriculum

The curriculum is designed to build skills in layers. You‚Äôll start with Linux and networking fundamentals, then move into core AWS services, automation, and a final capstone project.

Module 1
Linux & Command Line Foundations
  • Understanding the shell & basic commands
  • Filesystem navigation, permissions, users & groups
  • SSH, remote servers & simple shell scripts
Module 2
Networking Fundamentals for Cloud
  • IP addressing, subnets & routing basics
  • DNS, HTTP/HTTPS & load balancing concepts
  • How these map into cloud networks
Module 3
Core AWS Services (EC2, S3, IAM)
  • Launching and managing EC2 instances
  • Using S3 for storage and static websites
  • IAM users, groups, roles & policies
Module 4
VPC & Cloud Networking
  • Designing VPCs with public & private subnets
  • Route tables, internet & NAT gateways
  • Security groups and NACLs in practice
Module 5
Infrastructure as Code & Automation
  • Terraform basics: providers, resources, state
  • Creating and updating AWS resources via code
  • Intro to CI/CD for cloud infrastructure
Module 6
Monitoring, Logging & Troubleshooting
  • CloudWatch metrics, logs & alarms
  • Debugging common AWS issues
  • Best practices for reliability & cost
Capstone Project
Deploy a Production-Style 2-Tier Application on AWS

Design and deploy a simple, secure web application with a database backend using VPC, EC2, load balancing, and monitoring. Use Terraform where possible, and present your architecture and decisions at the end of the program.
