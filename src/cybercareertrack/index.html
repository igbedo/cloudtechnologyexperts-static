Program Structure

Cybersecurity Career Track Curriculum

The curriculum is split into modules that build from fundamentals to practical SOC operations. You‚Äôll learn the ‚Äúwhy‚Äù behind attacks and the ‚Äúhow‚Äù of responding to them.

Module 1
Cybersecurity & SOC Foundations
  • Security principles: CIA triad, risk, threats & vulnerabilities
  • What is a SOC? Roles, tiers & responsibilities
  • Overview of common security tools and log sources
Module 2
Operating Systems, Network & Identity Basics
  • Windows & Linux basics from a security perspective
  • Network protocols, ports, and common attack surfaces
  • Authentication, authorization & identity concepts
Module 3

  • Phishing, ransomware, brute force & web attacks

  • Understanding Windows, Linux, firewall & application logs

  • Deep-dive investigations using multiple log sources

  • Incident response workflow & communication
Module 6

  • Hardening basics for endpoints, identity & cloud
  • Using MITRE ATT&CK to find detection gaps
  • Working with playbooks & continuous improvement
Capstone Project
End-to-End SOC Investigation & Case Report

Work through a simulated multi-stage attack, from initial alert to containment and lessons learned. Analyze logs in a lab SIEM, map the activity to MITRE ATT&CK, and produce a professional incident report you can show to employers.
